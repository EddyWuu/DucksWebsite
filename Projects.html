An automated greenhouse monitoring and control system that integrates sensor data to manage watering, temperature, and humidity levels. The system supports both preset and adaptive watering modes, using capacitance sensors to optimize plant hydration.

Spritfill

A pixel art/sprite creator and number-filling app that lets users design and color sprites using predefined or custom color palettes. Users can import real-life images, which are processed into pixel art, and then fill in pixels using a numbered color system. Features include canvas customization, sprite exporting, and a community tab for sharing creations.
